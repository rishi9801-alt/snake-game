# 🐍 Snake Game

A simple Snake Game built using HTML, CSS, and JavaScript.  
This project demonstrates DOM manipulation, game logic, and keyboard event handling.

---

## 🎮 Features

- Classic snake movement
- Food generation at random positions
- Score tracking system
- Game over detection (collision with wall/self)
- Smooth keyboard controls

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
